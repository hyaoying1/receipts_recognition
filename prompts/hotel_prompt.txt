请输出以下 JSON：
{
  "文件类型": "酒店水单",
  "确认号": 酒店确认号或预定号（如 Confirmation No. / Order No. / Reservation ID / CRS No. 等），无法确定时为 null,
  "入住日期": 入住日期，格式必须为 "YYYY-MM-DD"，无法确定时为 null,
  "离店日期": 离店日期，格式必须为 "YYYY-MM-DD"，无法确定时为 null,

  "币种": 币种代码（如 CNY、USD），无法确定时为 null
}

补充规则（很重要）：
一、酒店名字提取规则
优先从酒店 LOGO 下方提取酒店名称。
若 LOGO 下没有名称，则从账单正文（如抬头、酒店信息区、Hotel Name 字段）中查找酒店名称。
若仍无法确定，则 "酒店名字": null。

二、城市提取规则
城市必须从“地址（Address）”相关字段或完整地址文本中提取。
不能从酒店名字中推断城市（例如 “北京希尔顿酒店” 不代表城市是北京，必须以地址为准）。
若账单中没有可解析的地址，则 "城市": null。

三、日期规则
所有日期输出格式必须为 "YYYY-MM-DD"。
若票据中只有“月-日”，需尝试根据其他信息推断年份；若无法推断年份，则日期填 null。
"入住日期" 与 "离店日期" 都按此方式处理。

四、金额规则
金额必须为数字类型（如 123.45）。
若金额包含货币符号（￥ / $ / HKD 等），仅提取数字部分。
"总金额" 优先取账单中最明确的 Total / Amount Due / 合计 字段对应的金额。
